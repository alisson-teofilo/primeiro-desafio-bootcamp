git init : iniciar um novo repositório
git add . : "Prepara" todos os arquivos (que não estão listados no .gitignore) em todo o repositório.
git commit -m "anotações": Inicialica o commit com opção de comentar as alterações
git push origin main: "Empurra" as alterções local para o servidor.
git status: Mostra o status do arquivo de trabalho
git clone [url]: clona um repositório 
git status: Mostra o estado do arquivo
git diff: Mostra exatamente o que foi alterado 
git remote -v : exibe os repositórios remotos
git remote set-url origin "http://novourl": rediireciona o repositório para outro diretório
